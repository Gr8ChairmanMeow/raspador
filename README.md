# raspador #

## Full-Stack NYT Article Scraper ##
See what's going on in the world today!

* Mongoose/MongoDB
* HandlebarsJS
* ExpressJS
* Heroku-deployed

### Features include ###
1. Save articles
   * View them later!
1. Comments
   * Add comments
   * View comments
   * Edit comments
   * Delete comments
1. Scrape new articles
